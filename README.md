# Aha Interview - Backend
---
This is the interview project of the backend engineer position of Aha Entertainment. The project is based on the [Fastify](https://fastify.dev/) framework and [TypeScript](https://www.typescriptlang.org/). The project is a simple membership system, which can be used to register, log in, and query user information. The project uses [MySQL](https://www.mysql.com/) as the database, and use [Auth0](https://auth0.com/) as the third-party authentication service.

- API Document
  > In the project root directory, please open swagger.yaml to view the API document.
